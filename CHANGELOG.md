# ImgKIT Changelog

All notable changes and improvements across ImgKIT versions.

---

# v1.0
Initial working prototype.

## Features
- Image upload support
- Crop functionality
- Greyscale conversion
- Gaussian blur
- Brightness adjustment
- Contrast adjustment
- Export processed image

## GeoTIFF Support
- TIFF loading
- Rasterio fallback
- CRS extraction
- Latitude/Longitude metadata
- Multi-band raster support

---

# v2.0
Major feature upgrade and UI redesign.

## Added
- Resampling pipeline
- 6 interpolation methods
- Histogram visualization
- RGB channel toggles
- Stretch factor controls
- Bit depth conversion
- Improved landing page
- Better sidebar workflow
- Percentage-based crop sliders

## Improved
- UI readability
- Metadata presentation
- Histogram interaction
- Streamlit styling

---

# v2.1
Multi-output workflow implementation.

## Added
- 1–4 output processing
- Independent edit pipelines
- Parallel output visualization
- Individual downloads
- Session-state output management
- Color-coded output slots

## Improved
- Workflow flexibility
- Processing organization
- User interaction
- UI structure
